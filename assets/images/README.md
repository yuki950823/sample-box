## üìè Image Specifications

### General Guidelines
- **Format**: JPG for photos, PNG for graphics with transparency
- **Quality**: High quality (80-90% JPG compression)
- **Aspect Ratios**: Maintain recommended sizes for best display
- **File Size**: Optimize images (keep under 500KB per image)

### Optimization Tips
1. Use tools like TinyPNG or ImageOptim to compress images
2. Export at 2x resolution for retina displays if needed
3. Consider WebP format for better compression
4. Use descriptive filenames
